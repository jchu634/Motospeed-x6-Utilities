## Motospeed x6 Utilities
These are unofficial utilities developed using reverse engineered information.

### Battery Utility
Cross-platform system tray application which shows the battery level, (polled every minute).
- Note, has only been tested on Windows.
- I do not have any devices to test Linux + MacOS, but I don't see any reason why it shouldn't work.

### WebHID configurator
Cross-platform webHID configuration site that lets you modify your mouse settings.
- Note does not support macros.
- Available at [(http://x6-utility.keshuac.com/)](http://x6-utility.keshuac.com)
